# Potfolio App - Next.js 16 + TypeScript

Demo portfolio app for middle frontend-developer.

## Tech

- Next.js 16 (App Router)
- TypeScript (strict mode)
- Tailwind CSS
- Reack Hooks (useLocalStorage, useState, useEffect)
- Async server components

## Functionality

- Main page with product card and user profile (localStorage)
- Page `/posts` with posts (server fetch + client-side search)
- Typed components (Button, ProductCard, UserProfile, PostsList)
- Advanced TypeScript-utils (SnakeToCamel, DeepReadonly, NestedPath, AsyncState)

## How to run

```bash
npm i
npm run dev
```

### Eslint
